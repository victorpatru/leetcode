What does HTTP status 200 mean? #http-codes
---
OK — the request succeeded.
===

What does HTTP status 201 mean? #http-codes
---
Created — the request succeeded and a new resource was created.
===

What does HTTP status 204 mean? #http-codes
---
No Content — request succeeded; response body is intentionally empty.
===

What does HTTP status 301 mean? #http-codes
---
Moved Permanently — the resource has a new permanent URL.
===

What does HTTP status 302 mean? #http-codes
---
Found — temporary redirect; URL may change again.
===

What does HTTP status 304 mean? #http-codes
---
Not Modified — cached version should be used.
===

What does HTTP status 307 mean? #http-codes
---
Temporary Redirect — HTTP method must not change.
===

What does HTTP status 308 mean? #http-codes
---
Permanent Redirect — HTTP method must not change.
===

What does HTTP status 400 mean? #http-codes
---
Bad Request — server cannot process malformed request.
===

What does HTTP status 401 mean? #http-codes
---
Unauthorized — authentication required or failed.
===

What does HTTP status 403 mean? #http-codes
---
Forbidden — authenticated but not authorized.
===

What does HTTP status 404 mean? #http-codes
---
Not Found — resource does not exist.
===

What does HTTP status 405 mean? #http-codes
---
Method Not Allowed — endpoint does not support the request’s HTTP method.
===

What does HTTP status 409 mean? #http-codes
---
Conflict — request conflicts with current resource state.
===

What does HTTP status 410 mean? #http-codes
---
Gone — resource used to exist but was intentionally removed.
===

What does HTTP status 418 mean? #http-codes
---
I'm a Teapot — joke code from an RFC; sometimes used playfully.
===

What does HTTP status 429 mean? #http-codes
---
Too Many Requests — rate limit exceeded.
===

What does HTTP status 500 mean? #http-codes
---
Internal Server Error — generic server failure.
===

What does HTTP status 501 mean? #http-codes
---
Not Implemented — server doesn't support the requested method.
===

What does HTTP status 502 mean? #http-codes
---
Bad Gateway — upstream server sent an invalid response.
===

What does HTTP status 503 mean? #http-codes
---
Service Unavailable — server overloaded or down for maintenance.
===

What does HTTP status 504 mean? #http-codes
---
Gateway Timeout — upstream server didn’t respond in time.
===
